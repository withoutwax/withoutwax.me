# withoutwax.me

## Running Locally

```bash
$ git clone https://github.com/withoutwax/withoutwax.me.git
$ cd withoutwax.me
$ yarn
$ yarn dev
```

## Built Using

- [Next.js](https://nextjs.org/)
- [Vercel](https://vercel.com)
- [MDX](https://github.com/mdx-js/mdx)
- [Tailwind CSS](https://tailwindcss.com/)
